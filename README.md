# Sandbox Rights

[![Steam Workshop](https://img.shields.io/badge/Steam-Workshop-blue?style=for-the-badge&logo=steam)](https://steamcommunity.com/sharedfiles/filedetails/?id=3261510386)

![Icon](https://images.steamusercontent.com/ugc/2495636941502611953/01CEC670F165ED662AEFCAFCC7AF859136C7F2D3/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false)

**Sandbox Rights** is an addon designed to restrict the rights of all players on the server. It allows you to disable or enable specific sandbox activities for everyone.


## 🛠 Features

This addon provides granular control over the following actions:

* **Core Interactions:** Toggle Physgun, Gravity Gun, and Toolgun usage.
* **Inventory & Spawning:** Control Spawn Menu access and the ability to receive weapons.
* **Movement:** Manage Noclip permissions and enforce a **Max Speed** limit.
* **World Interaction:** Toggle item pickup, object damaging (breaking), and C-Menu (Property) access.
* **Management:** Built-in Whitelist and Ban system for specific players.


## ℹ️ Important Information
> [!IMPORTANT]
> All restrictions and bans **do not apply** to the server host (Listen Server host). The host always retains full access to all sandbox features.

> [!NOTE]
> Whitelisted players are stored in:  
> `\steamapps\common\GarrysMod\garrysmod\data\rights_whitelist.txt`

> [!NOTE]
> **Bans don't work on:** This system respects specific ULX ranks.

> [!CAUTION]
> Disabling "Damage objects" might interfere with certain game modes or specific contraptions that rely on entity health.
